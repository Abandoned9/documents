    JSR303数据校验

    DimitriYoon 7/26/2020 JavaCode

    我们在过滤请求的时候，除了在前端做一些拦截之外，在后端也应该规范相应的请求，而JSR303则能够 方便的帮助我们技术规范正式请求，它通常是配合yml或者yaml来食用

    # 最常见的普通参数属性验证

    在实体类上面通过 @Validated 注解来告诉SpringBoot这个实体类需要数据校验 然后在属性上通过注解来告诉我们的属性是什么格式的数据，这跟前端的正则表达式类似。 例如：

    import org.springframework.boot.context.properties.ConfigurationProperties;
import org.springframework.stereotype.Component;
import org.springframework.validation.annotation.Validated;

import javax.validation.constraints.Email;

@Component
@Validated
@ConfigurationProperties(prefix = "person")
public class Person {

    String name;
    int age;
    @Email
    String email;
    String phone;

}

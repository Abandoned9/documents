    # HashMap和Hash表的简介

    想要理解HashMap，首先需要了解Hash表这个数据结构

    ​ Hash表是一个非常经典的数据结构，它的核心是基于哈希值的桶 (Bucket) 和链表，他的核心就是一个一个的桶(Bucket)，桶是由数据转化成的Hash值来组成，底部是采用一个数组组成，这样做的好处就是在查询的时候速度非常快，能够达到O(1)级别。

    但是这样做会产生一个问题，就是在数据转化成的Hash值，在原来的数组中已经有了，那么这样就会产生一个冲突，Hash表是采用链表的形式来解决这个问题的，如果碰到了相同的Hash值，就采用头插法的方式，将新Hash值的尾部指向旧Hash值的头部，完成一个链表。

    举一个十分经典的例子：

    手机中的电话簿，例如我们现有联系人

    {张一，张二，张三，李一，李二，李三}

    而我们的电话簿一般都是按照名字的第一个字的拼音的首字母，按照字母表排序，比如张一的Hash值就为Z，但是这个时候我们还存入了一个张二，这个张二的Hash值呢，也为Z，这样两个值就产生了冲突，那么为了解决这个冲突呢，我们就将这个Hash值，做成一个链表，当遇到相同Hash为Z时，采用头插法，这样就解决了这个问题。

    在上面这个例子中，我们的Z，就叫做一个Hash桶，而出现两种Z的情况，就叫做碰撞，这就是一个完整的Hash表数据结构。

    如何在Java中实现Hash表，并且将它推广到所有元素？

    HashMap就做到了这一点

    int hashCode;
hashCode  → obj1 → obj2
    1
    2

    在上例的代码中，obj1或者obj2就是一个个的Hash桶，他的原理是将Java中的数据转化成一个hashCode，然后分别存储到一个个桶内。

    在Java中，hashCode是一个int类型的值，它大概有42亿多种可能，看起来非常多，但问题是，它始终是一个有限的值，当两个数据产生的hashCode相同时，如何处理这个问题？

    现在我们来看一下HashMap最基础的put方法：

        /**
     * Associates the specified value with the specified key in this map.
     * If the map previously contained a mapping for the key, the old
     * value is replaced.
     *
     * @param key key with which the specified value is to be associated
     * @param value value to be associated with the specified key
     * @return the previous value associated with <tt>key</tt>, or
     *         <tt>null</tt> if there was no mapping for <tt>key</tt>.
     *         (A <tt>null</tt> return can also indicate that the map
     *         previously associated <tt>null</tt> with <tt>key</tt>.)
     */
    public V put(K key, V value) {
        if (table == EMPTY_TABLE) {
            inflateTable(threshold);
        }
        if (key == null)
            return putForNullKey(value);
        int hash = hash(key);
        int i = indexFor(hash, table.length);
        for (Entry<K,V> e = table[i]; e != null; e = e.next) {
            Object k;
            if (e.hash == hash && ((k = e.key) == key || key.equals(k))) {
                V oldValue = e.value;
                e.value = value;
                e.recordAccess(this);
                return oldValue;
            }
        }

        modCount++;
        addEntry(hash, key, value, i);
        return null;
    }
